## 风险记录
- 当前只有视频素材，部分隐藏规则、数值和音效只能基于可见画面推断；PRD 中需标注“已观察”和“待确认”。
- 视频只展示了 `Gun` 被连续选择，`Laser` 与 `Beam` 的实际战斗效果未出现，PRD 采用名称和卡面做开发推断。
- 99+ 升级池超出视频展示范围，属于新增设计要求；后续开发需重点验证抽卡权重、卡池复杂度和 UI 可读性。
- 批量生成 99+ 卡图可能存在风格漂移，需通过统一提示词、manifest 和人工筛选控制一致性。
- 已发现并纠偏：先前生成图方向偏成外部飞机/舰队，不符合用户强调的“地球派出小飞机防御外星人”。这些图未进入项目。
- MusicGen 本地缓存不完整：`facebook/musicgen-small` 缺权重文件，`musicgen-medium` 有 `.incomplete`，当前无法离线生成 wav。

## 已修复
- 自动演示模式不符合用户要求：已删除自动演示按钮、自动选择逻辑和相关字符串。
- Level Up 卡片排版混乱：已通过截图检查重排标题、图标、说明和价格区域。
- 护盾图集裁切错误：原图集坐标会在地球旁边出现不自然半透明块，已改为 Canvas 实时能量环。
- 开局第一眼缺少小飞机信号：已加入 3 架基础近地防卫机，后续 Gun 升级继续扩编。
- 底部按钮区域被飞行物穿过显得混乱：已增加底部遮罩并提高按钮背景不透明度。
- 音效开启时缺少 `assets/audio/musicgen-earth-defense.wav` 导致 404：已补充本地合成循环 wav 文件。
- 升级卡图标使用大素材裁切导致残片和错切：已改为分类图标格映射，`Gun/Laser/Beam` 不再直接裁大飞机或激光特效素材。
- Beam 图标选中靠边格导致像被切掉：已换为居中的能量图标格。
- Canvas 文字排版和发光导致可读性差、文字出框：已将 HUD 与 Level Up 卡片迁移为 HTML/CSS，移除大面积文字发光。
- 美金符号不符合需求：已统一改为人民币符号 `¥`。
- 生图拆分素材仍有碎片和绿幕残边：已对飞机、敌人、子弹独立 PNG 做碎片清理和绿色残边清理，并增加贴图版本号防缓存。
- `4601` 与桌面 `PROJECT` 已有服务登记冲突：已迁移到下一个空闲登记端口 `4647`。
- 大图集裁切与模型伪透明背景导致重要元素仍不干净：已改为逐张单独生成黑底素材，本地透明化后接入 `assets/generated/individual/final/`，当前关键战斗元素不再使用旧图集裁切。
- 浏览器缓存导致升级文案仍显示旧版本：已更新 `index.html` 中 CSS/JS 版本参数，并给 `upgrades.js` 模块导入加版本号。
- 飞机不是俯视角：已重新生成 `topdown-plane-0/1/2/3` 与 `topdown-gun-icon`，当前运行时只采用俯视角飞机贴图。
- 升级缺少清晰进度反馈：已新增 HUD 内升级进度条。
- 旧音频层杂乱：已重写 Web Audio 引擎，降低总体增益，加入压缩器，重做音乐和音效。
- 画面不贴近视频中的密集环形弹幕：已新增金橙色外圈弹幕层，并调整中心地球构图。

## 新风险
- 当前 Three.js 引擎仍是第一版迁移，战斗动效已比 Canvas 版更稳，但爆炸、激光、能量束仍可继续精修。
- Google 图片模型输出不是真透明 PNG，而是黑底/棋盘格等视觉背景；后续继续生成素材时必须保留本地透明化检查，不可直接接入原图。
- 用户曾临时提供外部图片 API 配置，但后续明确要求不要再使用；项目文件不得保存该 Key，也不得把外部接口图集接入运行时。当前玩法增强图集已用本地程序化版本覆盖。

## 2026-05-02 新修复
- 外圈弹幕出现方块头和廉价细线：根因是上一版用 `PointsMaterial + LineSegments` 临时模拟，已改成透明弹体 Sprite。
- 敌人像单张静态图片漂过来：根因是敌人对象只用单个 Sprite 平移，已改成 Three.js 组合体并加入旋转能量环、核心光效、脉冲尾焰和光晕。
- 升级选择缺少对应表现：已在选择 `Gun/Laser/Beam` 后触发机群起飞、激光扫射、宽幅能量束。
- 自生成音效质量差：当前运行时已切换到 Kenney Sci-fi Sounds CC0 音效包，自生成 SFX 仅作为历史文件保留但不再引用。
- MusicGen 本地缓存缺权重：已允许脚本补齐 Hugging Face 模型文件，并成功生成 `assets/audio/musicgen-earth-defense.wav`。
- 用户希望不用等待 Level Up 也能购买升级：已新增战术商店，商店购买不推进普通 Level Up 触发进度，避免破坏原有节奏。
- 后期只有金币、玩法单一：已新增 10 关 20 波结构，第 20 波 Boss 与 Boss 击杀后的核心升级选择，后期有更明确的目标与节奏。
- Boss 不能是一枪即死：已给 10 个 Boss 配置递增血量，首关 Boss 基础血量 595，十关 Boss 基础血量 11620，并随已有升级数额外加成。
- 重要角色不能再依赖旧裁切图：已逐张生成 3 种玩家飞机和 3 种敌人角色，单独透明化后替换运行时贴图。
